## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd backend/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

### Backend

`mongo` - database

`port` - running port



`CURRENT_ENVIRONMENT`

`SMTP_USERNAME`

`SMTP_PASSWORD`

`SMTP_PORT`


### Frontend


`PORT`
